Car Shopping List Project

Overview

This project showcases a user-friendly car shopping list page, where users can effectively select, filter, and save cars they are interested in, inspired by features from Rotten Tomatoes, Yelp, and Goodreads.

Key Features and Comparison

The table below highlights the adapted features from each inspiration source:

Feature Rotten Tomatoes Goodreads Yelp
Sort by Rating Yes Yes No
Filter by Category Yes No Yes
Different Views No Yes No
Smart Recommendations Yes Yes No
Reset Filters Yes No No

Implementation Details

Implemented a multi-filter interface allowing users to sort cars by type, color, and price, complemented by a user-friendly card view displaying detailed information at a glance.

  • Car Type, Color, and Price Filters
  • Reset Filter for returning to initial selections
  • Add/Save functionality for user preferences

Link to the live site: View Car Shopping List

My Role in the Project

As the lead developer, I was responsible for the entire front-end development and ensuring that the interface is intuitive and meets user expectations based on the insights gained from the compared platforms.
